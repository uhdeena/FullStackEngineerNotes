Selectors

  1. Selector Speceficity - Ranking System

    1. #id-name
    2. .class-name
    3. element

  2. Chaining Selectors - Must match all criteriea.

    3. Example

    • HTML - <h3 section="heading">
    • CSS - h3 .section-heading { }

  3. CSS as HTML - Use <style> tags and CSS syntax.

  4. Inline Styles - Added to HTML element 'style' attribute in opening tag. Seperate each with a semicolon (;).

  5. CSS Descendant Selector - Use spaces between selectors.

  6. Groups of CSS Selectors - Separate with a comma ( , ).

  7. CSS ID Selector - Matches ID attribute

    8. Example

    • HTML with this code
      <h3 ID="test">
      would need to be used in CSS like this:
      #test {
      }

  8. CSS Class and ID Selectors - Classes are .class(reusable), #id(not reusable).

  9. Type Selectors - Match ALL elements of a type or tag name.

  10. 'Style' - An attribute for adding CSS directly.

    11. Example

    • <p style="color: green">Test Paragraph</p>

  11. Descendant Selector - Matches elements that are descended from another matched element. Denoted by a space between each.

    12. Example

    • div p {
      }

How to Write or Link

  1. CSS in seperate .CSS file, add to code with:

    2. Example

    • <head>

      <link href="style.css" type="text/css" rel="stylesheet">

       </head>>

  2. Write in HTML File

    3. Example

    • <head>

      <style>

       h1 {
      color: blue;
      }

      </style>

      </head>

  3. HTML attributes with multiple attributes should be seperated by a space.

Visual Rules

  1. Web Safe Fonts - Supported across omst browsers and operating systems.

    • Font must be donloaded on users computer or with the site.
    • May not appear if not web safe.
    • If typeface is more than one word, put in quotes.

    Example

    • h1 {
      font-family: 'Courier New'
      }

  2. Font-weight - Controls how BOLD or THIN or NORMAL

  3. Text Align - Aligns text to element thatholds it. AKA its 'parent'.

  4. Color

    • Background - written as background-color:
    • Foreground - written as color:

  5. Opacity - The measure of how transparent an element is from 0%-100%

    6. Example

    • h1 {
      opacity: 0.5;
      }
